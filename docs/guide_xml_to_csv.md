# Data Extraction & Conversion Guide
> **From MediaWiki to Graph Data (CSV)**

본 문서는 KMUA 위키 시스템에 구축된 데이터를 XML로 추출(Export)하고, 파이썬 파서를 통해 **지식 그래프용 CSV로 변환**하는 전체 과정을 설명합니다.

---

## 1. MediaWiki에서 XML 추출하기 (Export)

위키에 입력된 문서들을 하나의 파일로 내려받는 과정입니다.

### 1-1. 내보내기 페이지 접속
1. 위키 사이트에 로그인합니다.
2. 좌측 메뉴의 **[특수 문서(Special Pages)]**를 클릭하거나, 주소창 뒤에 `Special:Export`를 입력하여 접속합니다.
    * 예: `http://kmua-wiki.org/index.php/특수:내보내기`

### 1-2. 문서 선택 및 다운로드
1. **[빈칸에 문서 제목 추가]** 란에 추출하고 싶은 문서 제목을 입력합니다.
    * **Tip:** 분류(Category)를 이용하면 편리합니다. `분류:신축공사개요`를 입력하고 [추가] 버튼을 누르면 해당 분류의 모든 문서가 자동으로 리스트에 들어갑니다.
2. **[옵션]** 설정:
    * **현재 판만** (체크 권장: 과거 수정 기록까지 다 받으면 파일이 너무 커집니다.)
    * **틀 포함** (체크 해제: 데이터만 필요하므로 체크하지 않습니다.)
    * **파일로 저장** (반드시 체크)
3. **[내보내기]** 버튼 클릭
4. 다운로드된 파일(예: `Wikipedia-20260105.xml`)을 확인합니다.

---

## 2. 데이터 변환 (XML → CSV)

내려받은 XML 파일을 연구용 CSV 데이터로 변환하는 과정입니다.

### 2-1. 파일 이동
다운로드한 XML 파일을 프로젝트 폴더 내 **입력 전용 폴더**로 이동시킵니다.
* **위치**: `kmua-construction-summary/data/03_raw_xml/`
* *주의: 파일명은 한글/영어 상관없으나, 확장자는 반드시 `.xml`이어야 합니다.*

### 2-2. 변환 프로그램 실행
1. 터미널(PowerShell 또는 CMD)을 엽니다.
2. 프로젝트 폴더로 이동한 뒤, 아래 명령어를 입력합니다.

```bash
# 파서 실행 명령어
python src/02_graph_construction/wiki_parser.py
